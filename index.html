FAQ

Preguntas frecuentes

¿Necesito tener Minecraft instalado?

No. El launcher descarga e instala Minecraft, Fabric y los mods automáticamente. Solo necesitas una cuenta de Minecraft Java Edition (Microsoft) para iniciar sesión.

¿Necesito Java?

Lo recomendable es Java 21. Si no lo tienes, el launcher detecta su ausencia y te enlaza a Adoptium (la versión open-source de OpenJDK). Lo instalas, abres el launcher otra vez y ya está.

¿Por qué Windows dice que es peligroso?

Porque el .exe no está firmado con un certificado comercial (cuesta cientos de euros al año). Windows desconfía por defecto. Pulsa "Más información → Ejecutar de todas formas" y aprueba el binario. Pasa una vez y ya no vuelve a salir.

¿Mis cartas se guardan en la nube?

El catálogo de cartas sí (para que pueda añadir nuevas sin tener que actualizar el launcher). Tu inventario personal se guarda local en tu PC, firmado criptográficamente para que nadie pueda darte cartas falsas editando un JSON.

¿Es de código abierto?

Sí. Todo el código del launcher está en GitHub: kperegrin/mi-servidor-launcher. Es un fork de HMCL (que también es open source) adaptado para BarrilMC.

¿Lo voy a poder usar en Linux/Mac?

Sí, hay un .jar y un .sh en cada release. Necesitas Java 21 instalado. El .exe es solo para Windows.

¿El launcher manda datos a alguien?

Solo lo necesario para que funcione: tu cuenta a Microsoft para autenticarte, mensajes al chat de Firebase si lo usas, y descargas de mods desde GitHub Releases. Nada más. Ningún tracking, ningún analytics.
